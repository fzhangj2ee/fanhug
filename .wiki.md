# Project Summary
The project is an educational video game aimed at children aged 8-10 to facilitate step-by-step learning of Chinese. It features interactive game modes that enhance vocabulary acquisition, character recognition, and sentence construction, employing a progressive learning approach to create an engaging and effective educational experience.

# Project Module Description
- **Vocabulary Match**: Match Chinese characters with their English meanings.
- **Character Quiz**: Identify the correct Chinese character for given English words.
- **Sentence Builder**: Construct Chinese sentences by arranging provided words, with improved functionality for the reset button.
