# Project Summary
PayApp is a secure payment web application that enables seamless payment processing and live sports betting. Users can register and log in using traditional methods or OAuth2 social media accounts, ensuring an intuitive experience. The platform integrates Stripe for secure transactions and The Odds API for real-time betting data. Key features include automatic grading of bets, real-time updates on outcomes, and easy access to betting functionalities.

# Project Module Description
- **Authentication System**: 
  - Supports email/password and OAuth2 social login (Google, Facebook).
  - Manages sessions and protected routes.
  - Allows user profile management.

- **Payment Processing**: 
  - Features a payment button on the landing page.
  - Integrates with Stripe for secure transactions.
  - Provides transaction history and confirmation.

- **Live Betting**: 
  - Offers real-time odds and score updates.
  - User-friendly interface for placing bets.
  - Automates grading of bets.

# File Description Inventory
- **README.md**: Project details and setup instructions.
- **AUTHENTICATION_SETUP.md**: Guide for setting up authentication with Supabase.
- **index.html**: Main HTML file for the React application.
- **package.json**: Lists project dependencies and scripts.
- **src/**: Contains all source code, including components, contexts, and pages.
- **public/**: Holds static assets like the favicon and robots.txt.
- **src/pages/**: Various application pages including homepage, live betting, login, signup, My Bets, and Wallet.
- **src/lib/**: Library functions for API integrations, including Supabase and The Odds API.
- **src/contexts/**: Context providers for managing application state.
- **src/components/**: UI components, including social login buttons and navigation bar.
- **src/hooks/**: Custom hooks for accessing authentication context.
