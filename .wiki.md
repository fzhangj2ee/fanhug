# Project Summary
PayApp is a secure payment web application that enables seamless payment processing. It allows users to register and log in using traditional email/password methods or social media accounts (Google, Facebook, Twitter, LinkedIn). The application features a straightforward payment interface where users can make quick payments using Stripe integration. PayApp aims to provide a user-friendly experience while ensuring high-level security and efficient transaction handling.

# Project Module Description
- **Authentication System**: 
  - Email/password registration and login using Supabase Auth.
  - Social login options (Google, Facebook, Twitter, LinkedIn) with automatic session management.
  - Real-time auth state synchronization.
  
- **Payment Processing**: 
  - Landing page with a payment button.
  - Stripe payment integration for secure transactions.
  - Payment confirmation and transaction history.

# File Description Inventory
- **README.md**: Contains project details and setup instructions.
- **index.html**: The main HTML file that serves the React application.
- **package.json**: Lists project dependencies and scripts for building and running the application.
- **src/**: Contains all source code, including components, pages, and data.
- **public/**: Holds static assets like the favicon and robots.txt.
- **src/contexts/AuthContext.tsx**: Manages user authentication state with Supabase Auth.
- **src/lib/supabase.ts**: Contains Supabase client configuration and helper functions for authentication.
- **src/pages/**: Contains various pages for the application, including login, signup, dashboard, and payment.

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for fast development and optimized production builds.
- **Stripe**: Payment processing platform for secure transactions.
- **Supabase**: Backend as a service for authentication and database management.
