# Project Summary
PayApp is a secure payment web application designed for seamless payment processing and live sports betting. Users can register and log in using traditional email/password methods or social media accounts (Google, Facebook, Twitter, LinkedIn). The platform features an intuitive payment interface integrated with Stripe for efficient transactions. Recent enhancements include a localStorage-based architecture, replacing Supabase, and integration with The Odds API for real-time betting features, providing users with live odds data from major sportsbooks. The interface is designed to emulate a professional sportsbook style, displaying betting odds in an American format for improved usability.

# Project Module Description
- **Authentication System**: 
  - Email/password registration and login.
  - Social login options with automatic session management.
  - Real-time auth state synchronization with enhanced error handling.

- **Payment Processing**: 
  - Landing page with a payment button.
  - Stripe integration for secure transactions.
  - Payment confirmation and transaction history.

- **Live Betting**: 
  - Real-time odds updates every 60 seconds with visual indicators.
  - Live score updates for active games.
  - User-friendly interface for placing bets on live sports.
  - Integration with The Odds API for fetching real-time odds data.
  - Displays Spread, Total, and Moneyline in American format with tabbed navigation.
  - Redesigned GameCard to match a professional sportsbook layout.
  - Dynamic breadcrumb navigation based on selected sport.
  - **Live Odds Update Section**: Added to the bet slip, showing real-time odds changes for selected bets with visual indicators.
  - **Category Filter Row**: New feature under "Today" section allowing users to filter displayed games by sport category (NFL, NBA, NHL, MLB, Soccer).

# Technology Stack
- **React**: Frontend library for building user interfaces.
- **TypeScript**: Superset of JavaScript for type safety.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Vite**: Build tool for fast development and optimized production builds.
- **Stripe**: Payment processing platform for secure transactions.
- **React Query**: For managing server state and caching.
- **The Odds API**: For fetching real-time sports odds from multiple sportsbooks.
