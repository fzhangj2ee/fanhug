# Project Summary
FanHug is a secure payment web application designed for seamless payment processing and live sports betting. The platform provides an intuitive user interface for registration and login, supporting both traditional methods and OAuth2 for social media accounts. By integrating with Stripe for secure transactions and The Odds API for real-time betting data, FanHug enhances the betting experience with features like automatic grading of bets, real-time updates, and an admin dashboard for user management. Recent updates have focused on improving the password reset functionality and enhancing user tracking for admin features.

# Project Module Description
- **Authentication System**: 
  - Supports email/password and OAuth2 social login (Google, Facebook).
  - Manages sessions and secured routes.
  - Enables user profile management with improved error handling.

- **Payment Processing**: 
  - Features a payment button on the landing page.
  - Integrates with Stripe for secure transactions.
  - Provides transaction history and confirmations.

- **Live Betting**: 
  - Offers real-time odds and score updates.
  - User-friendly interface for placing bets.
  - Automates grading of bets.

- **User Profile Management**: 
  - Accessible from the user icon dropdown.
  - Displays and allows editing of user information.
  - Shows account details and betting statistics.

- **Admin Dashboard**:
  - Visible only to designated admin users.
  - Lists all registered users with detailed views including win/loss statistics and current bets.
  - Provides functionality to manage user data and betting activity.

- **Password Reset Flow**:
  - **Reset Password Page**: Allows users to request a password reset email.
  - **Update Password Page**: Enables users to set a new password after clicking the reset link.
