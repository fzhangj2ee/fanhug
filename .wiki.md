# Project Summary
FanHug is a secure payment web application designed for seamless payment processing and live sports betting. It provides an intuitive user interface with options for registration and login, including traditional methods and OAuth2 for social media accounts. The platform integrates with Stripe for secure transactions and The Odds API for real-time betting data. Key functionalities include automatic grading of bets, real-time updates, an enhanced betting experience, and an admin dashboard for user management. Recent updates have improved the user interface and fixed critical issues, enhancing overall usability.

# Project Module Description
- **Authentication System**: 
  - Supports email/password and OAuth2 social login (Google, Facebook).
  - Manages sessions and secured routes.
  - Enables user profile management.

- **Payment Processing**: 
  - Features a payment button on the landing page.
  - Integrates with Stripe for secure transactions.
  - Provides transaction history and confirmations.

- **Live Betting**: 
  - Offers real-time odds and score updates.
  - User-friendly interface for placing bets.
  - Automates grading of bets.

- **User Profile Management**: 
  - Accessible from the user icon dropdown.
  - Displays and allows editing of user information.
  - Shows account details and betting statistics.

- **Admin Dashboard**:
  - Visible only to designated admin users.
  - Lists all registered users with detailed views including win/loss statistics and current bets.
  - Provides functionality to manage user data and betting activity.
