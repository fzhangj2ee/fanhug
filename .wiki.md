# Project Summary
FanHug is a secure payment web application designed for seamless payment processing and live sports betting. It offers an intuitive user experience with registration and login options, including traditional methods and OAuth2 for social media accounts. The platform integrates with Stripe for secure transactions and The Odds API for real-time betting data. Key features include automatic grading of bets, real-time updates, enhanced betting functionalities, and OAuth authentication. Recent enhancements have improved odds calculations and user interface elements, including the Bet Slip styling to align with branding. The left sidebar has been streamlined to focus on key navigation options.

# Project Module Description
- **Authentication System**: 
  - Supports email/password and OAuth2 social login (Google, Facebook).
  - Manages sessions and secured routes.
  - Enables user profile management.

- **Payment Processing**: 
  - Features a payment button on the landing page.
  - Integrates with Stripe for secure transactions.
  - Provides transaction history and confirmations.

- **Live Betting**: 
  - Offers real-time odds and score updates.
  - User-friendly interface for placing bets.
  - Automates grading of bets.

- **User Profile Management**: 
  - Accessible from the user icon dropdown.
  - Displays and allows editing of user information.
  - Shows account details and betting statistics.
